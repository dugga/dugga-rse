Spooled File Conversion Types

Identifier

com.softlanding.spooledfiles.save

Description

This extension point is used to plug in alternate file types for saving spooled files.

Markup

   <!ELEMENT extension (type+)>
   <!ATTLIST extension
     point CDATA #REQUIRED
     id    CDATA #IMPLIED
     name  CDATA #IMPLIED
   >

  • point -
  • id -
  • name -

   <!ELEMENT type EMPTY>
   <!ATTLIST type
     id               CDATA #REQUIRED
     class            CDATA #REQUIRED
     name             CDATA #REQUIRED
     file_extension   CDATA #REQUIRED
     prtdevtypefilter (*SCS|*IPDS|*USERASCII|*AFPDS|*LINE) #IMPLIED
   >

  • id - a unique name that will be used to reference this type
  • class - a fully qualified name of the Java class that implements com.softlanding.spooledfiles.ISaveToFile interface
  • name - a translatable name that will be used for presenting this type in the UI
  • file_extension - the default file extension for this type
  • prtdevtypefilter - the optional printer device type for which this conversion type is available

Example

The following is an example of the extension point usage: 

 <extension point="com.softlanding.spooledfiles.save">
  <type
   id="com.example.xyz.type1"
   name="Glimwitz File"
   class="com.example.xyz.SaveToGlimwitz"
   prtdevtypefilter="*SCS"
   file_extension="gwz">
  </type>
 </extension>

API Information

Plug-ins that want to extend this extension point must implement com.softlanding.spooledfiles.ISaveToFile interface.

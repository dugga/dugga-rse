
COMPLETING THE INSTALLATION MANUALLY

--------
RDi 7.5
--------

The IBM i Messages subsystems and the IBM i Spooled Files subsystems 
that the RSE Extensions contribute do not get automatically added to 
existing RSE connections.  In order to add these subsystems to existing 
connections you must do so manually from the RSE Extensions Preferences:

Window -> Preferences -> Remote Systems -> IBM i -> RSE Extensions
